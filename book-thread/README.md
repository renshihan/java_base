3.producer-consumer模式
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    producer是生产者的意思，指生成数据的线程。
    consumer是消费者的意思，指使用数据的线程。

    producer-consumer模式在生产者和消费者之间加入了一个桥梁角色。
这个桥梁角色用于消除线程间处理速度的差异。
